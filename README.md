# Swiggy Insights Dashboard

This project presents a **Swiggy Insights Dashboard** built using Tableau, analyzing over **197K orders** to uncover customer behavior, sales performance, top restaurants, popular dishes, payment patterns, and state-wise sales contribution across India.

## Key Highlights
- **Monthly Sales Trend** visualized with interactive line charts.
- **State Sales Contribution Map** showing high-performing regions.
- **Top 10 Restaurants & Top 10 Dishes** ranked by orders.
- **Payment Mode Distribution** (UPI, Cards, COD, Wallets).
- **City-wise Sales Treemap** for quick comparison.
- **Customer Demographics** including age groups and Veg vs Non-Veg share.
- Multi-page, clean, and interactive Tableau visual storytelling.
